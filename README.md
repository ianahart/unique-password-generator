### Description
This password generator takes in and validates the user's preferred password length. It then
asks a few questions on what to include in the password. This includes lowercase characters, uppercase characters, numbers, and special characters.

### Installation
 ```git clone``` https://github.com/ianahart/unique-password-generator.git

### Usage
Open up the **index.html** in the root of the folder
and click **Generate Password** and follow the prompts.
